# RvYetRusFull
Русская локализация текста и текстур + шрифт для игры Rv there yet

## Что переведно:
- ✅ текст
- ✅ текстуры

## Дополнительно
Установлен шрифт для русской локализации

## Установка
Перейти в папку с игрой и перетащить папку Ride в папку игры

## Подробная установка:
В архиве в конечной папке будет 9 файлов

<br>-----------------------------------------------------------<br/>
<br>RvFont_P.pak<br/>
<br>RvFont_P.ucas   эти 3 файла отвечают за шрифт<br/>
<br>RvFont_P.utoc<br/>
<br>-----------------------------------------------------------<br/>
<br>RVThereYet_L_P.pak<br/>
<br>RVThereYet_L_P.ucas  эти 3 файла отвечают за перевод текста<br/>
<br>RVThereYet_L_P.utoc<br/>
<br>-----------------------------------------------------------<br/>
<br>Rvyetrus_P.pak<br/>
<br>Rvyetrus_P.ucas эти 3 файла отвечают за перевод текстур<br/>
<br>Rvyetrus_P.utoc<br/>
<br>-----------------------------------------------------------<br/>

<br>Все эти файла (смотря что нужно, можно и все) перетащить в папку по этому пути:
"(тут ваш диск и папка стима D:\Steam)\steamapps\common\Ride\Ride\Content\Paks"<br/>

Задать вопросы можно в этом руководстве стим - https://steamcommunity.com/sharedfiles/filedetails/?id=3600646118
