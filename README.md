# RvYetRusFull
Русская локализация текста и текстур + шрифт для игры Rv there yet

Что переведно:
+текст
+текстуры

Дополнительно установлен шрифт для русской локализации

Установка
Перейти в папку с игрой и перетащить папку Ride в папку игры

Подробная установка:
В архиве в конечной папке будет 9 файлов
---------------------------
RvFont_P.pak
RvFont_P.ucas   эти 3 файла отвечают за шрифт
RvFont_P.utoc
---------------------------
RVThereYet_L_P.pak
RVThereYet_L_P.ucas  эти 3 файла отвечают за перевод текста
RVThereYet_L_P.utoc
---------------------------
Rvyetrus_P.pak
Rvyetrus_P.ucas эти 3 файла отвечают за перевод текстур
Rvyetrus_P.utoc
----------------------------

Все эти файла(смотря что нужно, можно и все) перетащить в папку по этому пути "(тут ваш диск и папка стима D:\Steam)\steamapps\common\Ride\Ride\Content\Paks"
